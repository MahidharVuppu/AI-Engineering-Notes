# AI Engineering Notes

This repository contains my structured notes for *AI Engineering* by Chip Huyen.

The goal of this repo is to organize the main ideas from the book in a clean, reviewable format while preserving the practical engineering focus of each chapter.

These notes are meant to help me understand and revisit the core concepts behind building applications with foundation models.

## Book

**AI Engineering: Building Applications with Foundation Models**  
Author: Chip Huyen  
Publisher: O'Reilly Media

## Topics Covered

The notes follow the structure of the book and cover topics such as:

- Foundation models
- Large language models
- Multimodal models
- AI engineering vs traditional ML engineering
- Evaluation methodology
- AI system evaluation
- Prompt engineering
- Retrieval-augmented generation
- Agents
- Finetuning
- Dataset engineering
- Inference optimization
- AI application architecture
- Monitoring and user feedback
